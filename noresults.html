Search box instructions for use.

To use the search box, enter a single word which comes under one of the following:

  • Keyword, typically the name of the equation or law e.g. "stokes" to search for Stokes's Law, "newton" for Newton's Laws or "index" for Laws of Indices. Note:
    • There are no capital letters
    • There is no punctuation e.g. not " stokes' "
    • No superfluous uses of "s" e.g. not " newtons "
    • No plurals e.g. not " indices "
    • Correct spelling is used
  • The full name of a component of an equation e.g. for f=ma, type "force" and this equation will come up as a result. Note:
    • There are no capital letters
    • As force is in a lot of equations, there will be more than one result
  • Type in the equation e.g. f=ma. Note:
    • There are no capital letters
    • There are no spaces
    • Letters do not need a multiplication sign (*) between them
    • Use / for division where applicable
