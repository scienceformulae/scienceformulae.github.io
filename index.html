Use the search bar above to start exploring. Enter one of: astronomy, biology, chemistry, mathematics, physics and statistics.

Alternatively, try to find what you are looking for immediately by typing what you want in the search box.




Search box instructions for use.

To use the search box, enter one of the following:

  • Name of the equation e.g. newtons second law. Note:
    • There are no capital letters
    • There is no punctuation e.g. not newton's second law
    • Correct spelling is used
  • The full name of a component of an equation e.g. for f=ma, type "force" and this equation will come up as a result. Note:
    • There are no capital letters
    • As force is in a lot of equations, there will be more than one result
  • Type in the equation e.g. f=ma. Note:
    • There are no capital letters
    • There are no spaces
    • Letters do not need a multiplication sign (*) between them
    • Use / for division where applicable

The mission of this website is to help anyone revising GCSEs or A-levels, needing to find a formula that meets their requirements. For each formula, there is an explanation of how it works and many of the formulas are proven. Very little prior knowledge is assumed, so someone as young as eleven could grasp something as difficult as calculus, oblique collisions or calorimetry.
