Search box instructions for use.

To use the search box, enter a single word which comes under one of the following:

  • Keyword, typically the name of the equation or law e.g. "pythagoras" to search for Pythagoras's Theorem, "newton" for Newton's Laws or "index" for Laws of Indices. Note:
    • There are no capital letters
    • There is no punctuation e.g. not " pythagoras' "
    • No superfluous uses of "s" e.g. not " newtons "
    • No plurals e.g. not " indices "
    • Correct spelling is used
  • The full name of a component of an equation e.g. for f=ma, type "force" and this equation will come up as a result. Note:
    • There are no capital letters
    • Write trigonometric functions as their standard contracted name e.g. "sin", "cos", "tan" etc
    • As force is in a lot of equations, there will be more than one result

The mission of this website is to help anyone revising GCSEs or A-levels, needing to find a formula that meets their requirements. For each formula, there is an explanation of how it works and many of the formulas are proven. Very little prior knowledge is assumed, so a GCSE student could grasp something as difficult as calculus, oblique collisions or cosec.
